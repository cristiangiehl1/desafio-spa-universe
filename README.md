## 🔖 Abordagem

<p> Funcionalidades esperadas dos botões:</p>
<ul>

  <li>Stop: para o timer;</li>
  <li>Set: pode definir o timer;</li>
  <li>+: aumenta em mais 5 minutos o tempo do timer</li>
  <li>-: diminui em 5 minutos o tempo do timer. </li>
